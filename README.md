### mescyurasite

**Особисте портфоліо та сайт-резюме** на **React** і **TypeScript**, зібрані **Vite**. Є світла/темна тема, дві мови інтерфейсу (EN/UA), окремі сторінки портфоліо та контактів, кастомна 404 і футер із лайв-статусом Spotify (через Lanyard).

---

### Стек

- **UI**: React 19, TypeScript
- **Збірка**: Vite 6, `@vitejs/plugin-react`
- **Стилі**: Tailwind CSS v4 (`@tailwindcss/vite`)
- **Маршрути**: `react-router-dom`
- **Анімації**: `framer-motion`
- **Іконки**: `react-icons`
- **Інтеграції**: `use-lanyard` + WebSocket для Spotify у футері
- **UX**: `@tippyjs/react`, `react-tooltip`
